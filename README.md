# CalculadoraIA32

## Trabalho desenvolvido durante a disciplina de Software Básico

### Descrição

Calculadora simples desenvolvida em linguagem Intel Assembly x86.


### Funções:
- Soma
- Subtração
- Multiplicação
- Divisão
- Potenciação
- Módulo


Para compilar e linkar:

    make

Para executar:

    ./CALCULADORA

Para limpar os arquivos gerados:

    make clean
